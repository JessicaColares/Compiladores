Trabalho 08 - Otimização (parte 1)
Este trabalho pode ser feito individual ou em duplas.

Baseado nos slides, faça somente a integração das três técnicas de otimização apresentados: (i) simplificação de expressões; (ii) propagação de constantes e; (iii) eliminação de expressões comuns. 

Se você quiser, pode copiar os códigos disponibilizados. Se for o caso, faça ajustes, caso necessário. 

O código deve ler um arquivo com o código três endereços não otimizado e deve gerar um código otimizado. É necessário que o código seja feito em Python. 

Defina uma sequência de execução dos três otimizadores. 

Repita a execução dos três otimizadores até que nenhuma mudança seja feita. 

Entregue o código Python e pelo menos cinco arquivos de testes compactados em um arquivo ZIP.
