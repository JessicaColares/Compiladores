Trabalho 09 - Otimização (parte 2)
Este trabalho pode ser feito individual ou em duplas. Deixe claro no código Python (como comentário) os membros da equipe.

Baseado nos slides, gere um programa em Python que integre no trabalho anterior mais uma técnica de otimização, ou seja, dobrando constantes (tradução livre de constant folding). 

O código deve ler um arquivo com o código três endereços não otimizado e deve gerar um código otimizado. É necessário que o código seja feito em Python. 

Descreva em alto nível como será feita a tua solução. Pode ser anexado um arquivo adicional ou incluído como comentário no início do programa Python.

Defina uma sequência de execução dos, agora, quatro otimizadores. 

Repita a execução dos quatro otimizadores até que nenhuma mudança seja feita. 

Entregue o programa Python e pelo menos cinco arquivos de testes. Comprima os arquivos em um único arquivo ZIP.
